# TWO WORDS

**You got two words? I got a story. That's the deal. That's the whole deal.**

Listen. I've been in a lot of businesses. Spud guns. Milk steak. Rum ham. And I'm telling you right now, *this* is the one. This is the thing that changes everything.

You give it two words. Any two words. "Raccoon" and "mortgage." "Velvet" and "chainsaw." Doesn't matter. Doesn't matter one bit. You hit that button and GPT-4 — the smartest computer they got, I looked it up — it spits out a story so insane, so *beautifully unhinged*, that you will laugh until something comes out of your nose. Something will definitely come out.

I've shown this to people. Grown adults. They wept. One guy tried to invest. I said no. This one's mine.

---

## What It Does

- You type **two words**
- It generates a **ridiculous short story** using OpenAI GPT-4o
- The story types out on screen like it's being written *live* by a maniac
- There are example word pairs if you're too scared to think for yourself

That's it. That's the app. Beautiful in its simplicity. Frank Reynolds doesn't overcomplicate things.

---

## Stack

- **Next.js 14** — because it just works and I'm not explaining that further
- **OpenAI GPT-4o** — the big one, temperature cranked up, no guardrails on the *creativity*
- **Zero CSS frameworks** — raw inline styles, like God intended

## Setup

```bash
git clone https://github.com/Joshbrooks237/two-words.git
cd two-words
npm install
```

Create a `.env.local` and throw your OpenAI key in there:

```
OPENAI_API_KEY=your-key-here
```

Then:

```bash
npm run dev
```

Open `http://localhost:3000`. Type two words. Change your life.
